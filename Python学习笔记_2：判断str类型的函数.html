今天给大家介绍一下 python 中可以判断 str 类型的函数：

1、isdigit： 判断 str 是否为数字

2、isalpha：判断 str 是否为字母

3、isalnum：判断 str 是否为字母或数字

4、islower：判断 str 是否为小写

5、isupper：判断 str 是否为大写

6、istitle：判断 str 是否仅有首字母大写

7、isspace：判断 str 是否为空格符
